# SAASI demonstration: timed phylogenetic tree

A simulated birth-death-sampling tree with 13 tips. The tip states are
located separately in the
[demo_metadata](https://magpie-sfu.github.io/saasi/reference/demo_metadata.md)
file.

## Usage

``` r
demo_tree
```

## Format

An object of class `phylo`

## Examples

``` r
demo_tree
#> 
#> Phylogenetic tree with 13 tips and 12 internal nodes.
#> 
#> Tip labels:
#>   sp2, sp4, sp8, sp9, sp10, sp26, ...
#> Node labels:
#>   nd1, nd2, nd6, nd22, nd27, nd74, ...
#> 
#> Rooted; includes branch length(s).
```
